# React form

+ Форма (имя, фамилия, "отправить"/очистка формы)
+ Выбор языка (ru, en)
+ Проверка полей (ввод только букв ru/en)
+ Выбор стилей (default, dark, orange)

## Clone

`git clone https://github.com/tymino/form-validate-with-styles-localize.git`

## Setup

`yarn install`

## Run

`yarn start`
